Package is a foundation

            // bower.json
            {
                "name": "my-share",
                "version": "1.0.0",
                "main": ["my-share.js"]

Bower and format of bower.json

Component private HTML

            <a href="{{ href }}" class="my-share">
                <img src="{{ icon }}" class="my-share__icon"/>
                <span class="my-share__label">{{ label }}</span>
            </a>

This template is written using BEM Methodology.

Component interface

            <a class="my-share"
               data-href="{{ href }}"
               data-icon="{{ icon }}"
               data-label="{{ label }}"
            ></a>

This interface is close to the WebComponents.

WebComponents for comparison

            <my-share
               href="{{ href }}"
               icon="{{ icon }}"
               label="{{ label }}"
            ></my-share>

CSS for my-share Component

            .my-share__icon {
                vertical-align: middle;
                height: 16px;
            }
            .my-share__label {
                padding-left: 5px;
            }

We remain font-size and text-decoration untouched.

JS for my-share Component

            // Include all requirements
            var html = require('templates/my-share'),
                tpl = require('lodash-template'),
                domify = require('domify');

JS for my-share Component

            var template = tpl(html);
            function MyShare(el) {
                this.options = el.dataset;
                this.el = domify(template(this.options));
                this._bindEvents();
                el.parentNode.replaceChild(this.el, el);
            }
            module.exports = MyShare;
